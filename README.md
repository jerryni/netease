在入职的第一个星期, 主管会让每个新成员先完成这个<前端大作业>, 这边主要来记录一下;

这次完善了一下这个项目结构以及文档主要是因为, 我想用其他框架如reactjs, 来实现一套这样的作业, 让自己练手;
